Heart shape Detection script
First of all open python termianl run the following command

pip install -r requirement.txt

This line install all the desirecd pakages to run the script

after this open the project in the editor or run the code direct in the terminal.To run the code in the python terminal, the following command is

python Detector.py --image yourimagepath

if you want to use my images available with this project run the following command

python Detector.py --image DATA\\h7.png

or if you want to run the code in the any editor. So first go to edit configration and in there will be "parameter" text field write down the image path in that field and run Detector.py class

Constraints-:
always background of Heart image should be white
